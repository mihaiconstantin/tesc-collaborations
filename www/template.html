Data
The network graphs are based on the matrices provided by the ina package (i.e., https://github.com/mihaiconstantin/ina). Data used to build the matrices were collected using the web scraping tool TescPerf (i.e., https://github.com/mihaiconstantin/TescPerf). The web interface around TescPerf (i.e., TescPerfWeb - https://github.com/mihaiconstantin/TescPerfWeb) can be accessed at http://constantinmihai.com:100.

Dependencies
The following interface was built with qgraph (i.e., https://github.com/SachaEpskamp/qgraph) for plotting the network graphs and Shiny (i.e., https://shiny.rstudio.com/) for reactivity.
